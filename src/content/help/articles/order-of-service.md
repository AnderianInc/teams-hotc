# Order of Service

Plan every service as an ordered set of time slots, assign the right people to each one, and have those assignments flow into the calendar and each person's dashboard automatically.

## Concepts

- **Template** — a reusable blueprint (e.g. "Sunday 10am Main Service") with default slots like *Welcome, Worship Set 1, Announcements, Sermon, Response, Closing*.
- **Service** — a specific Sunday. Generated from a template, then edited freely without changing the template.
- **Slot** — one row on a service: title, duration, assignees.
- **Assignment** — a person on a slot. Creates a matching roster entry, so it shows on their calendar and they can accept/decline.

## Set up a template

1. Go to **Admin Panel → Order of Service → Templates**.
2. Click **New template**, give it a name and default start time.
3. Add slots in order with durations. Optionally pre-assign a default serving team per slot (e.g. Worship team for the worship set).

## Generate a Sunday's service

1. In **Upcoming Services**, click **New from template**.
2. Pick a template and the service date. The service is created with all the template's slots pre-filled.
3. A roster event is automatically created for that date and linked, so it appears on the roster calendar.

## Assign people

Open a service to see the run-sheet. For each slot:

1. Click **Assign**.
2. Choose a team, then pick a member from that team (or use **Search everyone** to browse the whole directory).
3. Add an optional role label (e.g. "Lead vocals").

The person gets a roster entry on the linked event and can accept or decline from their dashboard.

## Run-sheet view

Times are computed from the service start plus cumulative durations, so re-ordering or changing a duration re-flows the whole schedule. Use the **Print view** for the tech booth and worship pastor.
